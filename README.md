# Observer AI

The ultimate framework-agnostic visual feedback tool for AI Agents. 
A highly-optimized alternative to Agentation.

## Packages
- `@observer-ai/core`: Vanilla TS engine for UI highlighting and Markdown generation.
- `@observer-ai/web`: Web Component wrapper for Angular, Ionic, and Vanilla JS.
- `@observer-ai/react`: React wrapper for seamless Next.js and React integration.
- `@observer-ai/native`: React Native engine using native View measurements.
- `@observer-ai/mcp`: Local Model Context Protocol server for Cursor, Claude Code, and Antigravity.

## Installation
Run `npm install` from the root to install all workspace dependencies.
